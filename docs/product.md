# Product Definition - Coffee Roasting Manager

## 1. Vision
A precise, local-first web application designed for home coffee roasters to track, calculate, and manage their roasting sessions in real-time. The goal is to provide professional-grade metrics without the need for complex software or cloud dependencies.

## 2. Target Audience
- **Home Roasters**: Individuals using manual or semi-automatic roasters who want to digitize their roast logs.
- **Precision Seekers**: Users who care about Development Time Ratio (DTR), weight loss percentages, and consistent roast curves.

## 3. Core Features
- **Real-time Timer**: Track total roast time, drying phase, and Maillard phase.
- **Automatic Calculations**: Instant DTR (Development Time Ratio) and weight loss % calculations.
- **Milestone Tracking**: Easy buttons to record Dry End and First Crack.
- **Roast History**: A persistent log of all previous roasting sessions.
- **Recipe Management**: Create and follow step-by-step roasting plans.

## 4. User Goals
- "I want to save my roast data locally so I don't lose it if I clear my browser cache."
- "I want to see exactly when my roast is hitting 20% development time."
- "I want to be able to undo a deletion if I accidentally click the delete button."
